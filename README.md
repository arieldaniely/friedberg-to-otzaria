# הכי גרסינן לאוצריא

הפרויקט הזה ממיר את קובצי האקסל של חבילת "הכי גרסינן" מבית פרידברג לקובצי טקסט עם תגיות HTML בסיסיות, בפורמט שמתאים לטעינה ולעבודה בתוך [אוצריא](https://www.otzaria.org/).

הפרויקט הזה אינו יוצר מהדורה חדשה של החומר. הוא רק משנה את מבנה הקבצים והפורמט הטכני שלהם, כדי להפוך את החומר לנגיש יותר לשימוש אישי.

## מקור החומר

- אתר החומר המקורי: [Friedberg Genizah Project / הכי גרסינן](https://fjms.genizah.org/index.htm)
- עמוד אודות: [https://pr.genizah.org/](https://pr.genizah.org/)
- כל זכויות היוצרים בחומר המקורי שמורות לבעלי הזכויות של פרויקט פרידברג / הכי גרסינן.

## זכויות שימוש

- החומר המקורי אינו בבעלות הפרויקט הזה.
- ההמרה כאן נועדה לשימוש אישי בלבד.
- הפרויקט הזה מספק כלי המרה ואריזה מחדש של חומר קיים, בלי לטעון לבעלות על התוכן.
- לפני כל שימוש שאינו אישי, יש לבדוק את תנאי השימוש והזכויות מול האתר והגוף המקורי.

פירוט מלא מופיע גם בקובץ [LICENSE](C:/Users/kobi/Downloads/פרידברג/LICENSE).

## מה הפרויקט עושה

- קורא את קובצי ה־Excel מתוך תיקיית המקור `פרידברג/`
- מתעלם מקובצי המבוא, קבצי ה"סינופסיס לכל מסכת", וקובצי נעילה של Excel
- מאחד את דפי כל מסכת לקובץ טקסט אחד
- מוסיף כותרות `<h1>` ו־`<h2>`
- משווה כל נוסח ל־`וילנא`
- מסמן שינויים באמצעות `<b><i>...</i></b>`
- מפצל אוטומטית מקטעים חופפים לפי `סינופסיס מילים`

## שימוש מקומי

להתקנת התלות:

```bash
python -m pip install -r requirements.txt
```

להרצה על כל המסכתות:

```bash
python convert_friedberg.py --source .\פרידברג --output .\output
```

להרצה על מסכת אחת:

```bash
python convert_friedberg.py --source .\פרידברג --output .\output --tractate ברכות
```

ליצירת קישורי אוצריא מול טקסט Sefaria מקומי:

```bash
python generate_sefaria_links.py --friedberg-dir .\output --output .\output\links
```

כברירת מחדל הסקריפט מחפש את קובצי Sefaria תחת `ספריא/`, למשל `ספריא/סדר מועד/מגילה.txt`.
אם צריך להשתמש בתיקייה אחרת, ניתן להוסיף `--sefaria-dir <תיקייה>`.
אם לא נמצא קובץ מקומי למסכת, הסקריפט נופל חזרה ל־Sefaria API ולכן רק במקרה כזה דורש חיבור רשת בזמן הריצה.
ליצירת קישורים למסכת אחת בלבד, ניתן להוסיף `--tractate מגילה`; ללא `--tractate` הסקריפט מעבד את כל קובצי הפלט הנתמכים.
הוא מדלג על עוגנים קצרים מדי, כגון `גמ'` ללא טקסט נוסף, כדי להימנע מקישורים חלשים.
מספר הדילוגים נשמר בדוח ההתאמה תחת `skippedUnstablePairs`.

## פלט

- קובצי המקור נשארים בתיקיית `פרידברג/`
- הפלט נכתב לתיקיית `output/`
- כל מסכת נשמרת כקובץ `הכי גרסינן <שם מסכת>.txt`
- קובצי קישורים נכתבים לתיקיית `output/links/`
- דוחות התאמה נכתבים לתיקיית `output/links/reports/`

## הפצה אוטומטית

נוסף Workflow של GitHub Actions בקובץ [.github/workflows/release.yml](C:/Users/kobi/Downloads/פרידברג/.github/workflows/release.yml).

ה־workflow:

- מתקין את Python והתלויות
- מריץ את ההמרה
- יוצר קישורי `links/` מול Sefaria עבור כל המסכתות הנתמכות שנוצרו בפלט
- יוצר ZIP בשם `הכי גרסינן.zip`
- מפרסם GitHub Release עם הקובץ המצורף

ברירת המחדל כרגע היא גרסה `v0.1.0`.

ניתן להפעיל את ההפצה בשתי דרכים:

- `workflow_dispatch` עם ברירת מחדל `v0.1.0`
- `push` של תגית בפורמט `v*`
